What you need to install on your PC

1
Node.js — download and install from nodejs.org (choose the LTS version). This is the only thing you install. npm comes with it automatically.
2
Unzip the project, open a terminal in that folder, then run npm install — this downloads all the libraries automatically.
3
Copy .env.example to .env and fill in your keys. The demo below works with just ANTHROPIC_API_KEY.
4
Run npm start and open localhost:3000 — you're live. For Twilio webhooks use npx ngrok http 3000 to get a public URL.
